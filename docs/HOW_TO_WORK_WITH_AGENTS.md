# HOW_TO_WORK_WITH_AGENTS.md

This project is designed for agent-assisted development.

## Recommended Process

1. Use PLAN for larger tasks.
2. Use EXECUTE for scoped implementation.
3. Use REVIEW for audits and risk checks.

## Good Prompting Principles

- define scope
- define out-of-scope
- define whether the task is platform/shared/game-specific
- define constraints
- define validation
- prefer minimal safe solutions

## Important Rule

Do not ask agents vague questions without context.
And do not ask them to turn one game's assumptions into platform law unless that is explicitly the task.
