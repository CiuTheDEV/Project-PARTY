## First Priority

### Kalambury
The first main reference game module.

Likely characteristics:
- multiple modes,
- setup flow,
- game-owned session logic,
- possible additional device for secret/presenter content,
- timed rounds and scoring.

## Design Rule for All Games

Each game should define its own:
- win condition,
- round structure,
- mode structure,
- participant/device model,
- scoring model,
- special mechanics,
- and session flow.

The platform may help users discover and enter a game.
The platform does not define how every game must actually play.
