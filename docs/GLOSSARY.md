# GLOSSARY.md

## Hub
The homepage and game discovery layer of Project Party.

## Game Module
A self-contained game experience inside Project Party.

## Session
A game-owned play instance created inside a specific game.

## Shared UI
Reusable visual building blocks and design rules used across the platform.

## Game Registry / Metadata
The minimal information the hub needs in order to show, filter, and launch a game.

## Platform Layer
Shared code that belongs to the hub, routing, common UI, or general infrastructure.
